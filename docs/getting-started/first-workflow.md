# First Workflow

This page walks through the simplest way to use InfoDigest for the first time.

## 1. Sign in on the web app

Start in the web app. The dashboard is the main place where your saved content appears and where new items are added.

## 2. Add your first item

Use the Quick Collect box on the dashboard.

It accepts:

- a normal article URL
- a WeChat share link
- pasted share text that contains a WeChat URL
- plain text or Markdown

If you paste text instead of a URL, InfoDigest stores it as a note-like content item.

## 3. Open the saved item

After the item is created, open it from the dashboard list.

What you usually see next:

- the saved title
- source information when a URL was used
- reading status such as unread or reading
- summary content when available

## 4. Read actively

As you work through the item, use the reader to:

- highlight important passages
- add short notes
- tag the item for later filtering
- move it through your reading states

## 5. Come back later

After you build a library, use:

- [Read and Review](/workflows/read-and-review) to revisit important material
- [Knowledge Graph](/workflows/knowledge-graph) to understand how items connect

## When to switch to another collection method

- Use the [Browser Extension](/platforms/browser-extension) when a WeChat article is open in your browser.
- Use [Mobile Shortcut](/platforms/mobile-shortcut) when you want a lightweight mobile capture path.
