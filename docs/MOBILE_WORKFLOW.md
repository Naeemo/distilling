# 知萃移动端采集指南

手机微信内无法安装 Chrome 扩展，我们提供了多种替代方案来保存公众号文章。

## 方案对比

| 方案 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| 复制粘贴 | 所有平台 | 最简单，零配置 | 手动操作 |
| iOS 快捷指令 | iPhone/iPad | 一键分享，自动保存 | 仅 iOS |
| 邮件转发 | 所有平台 | 异步处理，跨设备 | 有延迟 |

---

## 方案一：复制粘贴（推荐）

最简单直接的方式，支持链接、文本、Markdown 和微信分享内容。

### 操作步骤

1. **打开微信公众号文章**
   - 点击右上角「···」
   - 选择「复制链接」或直接「分享到」浏览器

2. **在知萃 Web 端粘贴**
   - 访问 https://infodigest.app/dashboard
   - 在「快速采集」框中粘贴内容
   - 系统会自动识别类型并保存

### 支持的格式

#### 1. 网页链接
```
https://mp.weixin.qq.com/s/xxxxx
```
自动抓取文章内容。

#### 2. 微信分享文本
```
文章标题

https://mp.weixin.qq.com/s/xxxxx
```
自动提取链接并抓取内容。

#### 3. 纯文本/Markdown
直接粘贴任意文本内容，会保存为笔记。

## 方案二：iOS 快捷指令

一键分享，自动保存到知萃。

### 安装步骤

1. **下载快捷指令**
   - 在 iPhone/iPad 上打开 Safari
   - 访问: https://infodigest.app/shortcuts/save
   - 点击「获取快捷指令」

2. **配置 API 密钥**
   - 登录知萃 Web 端
   - 进入「设置 → 开发者 → API 密钥」
   - 复制密钥
   - 快捷指令首次运行时会要求输入

### 使用方法

1. 在微信中打开公众号文章
2. 点击右上角「···」→「分享」
3. 选择「知萃保存」快捷指令
4. 等待提示完成

### 快捷指令配置（供参考）

如需手动创建快捷指令：

```
1. 接收内容：从分享菜单
2. 如果输入包含 "mp.weixin.qq.com":
   - 提取 URL
   - 否则使用输入文本
3. 获取 URL 内容
4. 发送 POST 请求:
   - URL: https://api.infodigest.app/v1/contents/quick-add
   - Headers:
     - Authorization: Bearer [YOUR_API_KEY]
     - Content-Type: application/json
   - Body:
     {
       "url": "[文章链接]",
       "title": "[文章标题]",
       "source": "ios_shortcut"
     }
5. 显示通知: "已保存到知萃"
```

## 方案三：邮件转发

适合喜欢通过邮件稍后阅读的用户。

### 配置步骤

1. **获取专属邮箱地址**
   - 登录知萃 Web 端
   - 进入「设置 → 采集设置 → 邮件转发」
   - 复制您的专属邮箱地址（格式: `save-xxxx@infodigest.app`）

2. **将邮箱添加到通讯录**
   - 建议添加到常用联系人，方便转发

### 使用方法

1. 在微信中打开公众号文章
2. 点击右上角「···」→「邮件发送」
3. 收件人填入您的专属邮箱
4. 发送邮件

### 邮件处理规则

- **邮件标题**: 可选，留空则自动提取文章标题
- **邮件正文**: 第一行 URL 会被识别，其余内容作为笔记
- **处理时间**: 约 1-5 分钟
- **失败通知**: 处理失败会收到回复邮件

### 示例邮件

```
To: save-xxxx@infodigest.app
Subject: 我的笔记标题（可选）

https://mp.weixin.qq.com/s/xxxxx

这里可以添加额外的笔记内容...
```

## 常见问题

### Q: 粘贴微信链接后没有自动抓取？
A: 检查链接是否完整（以 `https://mp.weixin.qq.com` 开头）。如果仍有问题，可能是文章被微信限制访问，可以尝试复制全文粘贴。

### Q: 快捷指令提示"未授权"？
A: 检查 API 密钥是否正确配置。可在知萃 Web 端的「设置 → 开发者」中重新生成密钥。

### Q: 邮件转发后多久能看到？
A: 通常 1-5 分钟。如果超过 10 分钟仍未出现，请检查垃圾邮件文件夹是否有处理失败通知。

### Q: 安卓手机有什么方案？
A: 安卓手机推荐使用「复制粘贴」方案，或使用支持分享菜单的浏览器（如 Chrome）打开文章后直接复制链接。

## 技术细节

### API 端点（供开发者）

```
POST /api/v1/contents
Content-Type: application/json
Authorization: Bearer {token}

# 创建链接内容
{
  "url": "https://mp.weixin.qq.com/s/xxxxx"
}

# 创建文本内容
{
  "title": "自定义标题",
  "contentText": "文本内容...",
  "sourceType": "MANUAL"
}
```

### 微信文章抓取限制

- 需要文章处于公开访问状态
- 部分付费/原创声明文章可能无法完整抓取
- 建议在文章发布 24 小时内采集，避免链接过期
