# Web App

The web app is the main InfoDigest experience.

## What you do there

Use the web app to:

- sign in and manage your session
- collect links and pasted text
- browse your distilled feeds
- create and edit workspace articles
- open the reader
- review past material
- explore the knowledge graph

## Main screens

### Landing page

The landing page introduces the product and links into sign-in and registration.

### Feeds

Feeds is your distilled library home. It is the fastest way to:

- add content
- filter by reading state
- search saved items
- reopen something you were already reading

### Workspace

The workspace is where you actively use information instead of only storing it.

It lets you:

- start from an initial idea
- pull in references from your saved library
- get an AI-generated draft article
- keep editing the draft yourself

### Reader

The reader is where you spend time with a saved item instead of just filing it away.

### Review

The review page turns saved material into recall practice.

### Knowledge graph

The graph views help you move from isolated reading to connected understanding.

## When the web app is enough on its own

The web app is the best default if you:

- mainly save normal web links
- want one place for reading and organization
- do not need extension-based extraction
