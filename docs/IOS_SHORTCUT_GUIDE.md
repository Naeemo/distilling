# iOS 快捷指令安装指南

## 快速安装

1. **在 iPhone/iPad 上点击以下链接：**
   ```
   https://www.icloud.com/shortcuts/xxx（待发布）
   ```

2. **获取 API 密钥：**
   - 访问 https://infodigest.app
   - 登录您的账户
   - 进入「设置 → 开发者」
   - 点击「生成 API 密钥」
   - 复制密钥

3. **配置快捷指令：**
   - 首次运行时会提示输入 API 密钥
   - 粘贴您复制的密钥
   - 选择「始终允许」

## 使用方法

### 保存微信公众号文章

1. 在微信中打开公众号文章
2. 点击右上角「···」
3. 选择「分享」
4. 在分享菜单中找到「知萃保存」
5. 等待通知显示「内容已保存成功」

### 保存任意网页

1. 在 Safari 中打开网页
2. 点击分享按钮（底部的方块箭头）
3. 选择「知萃保存」
4. 等待保存完成

### 保存选中的文本

1. 选中想要保存的文本
2. 点击「分享」
3. 选择「知萃保存」
4. 内容将以笔记形式保存

## 故障排除

### 提示"未授权"

- 检查 API 密钥是否正确
- 重新生成 API 密钥并更新快捷指令

### 提示"无法连接"

- 检查网络连接
- 确认可以访问 https://infodigest.app

### 找不到快捷指令

- 确保快捷指令已安装
- 在「设置 → 快捷指令」中启用「允许不受信任的快捷指令」

## 手动创建（备选）

如果一键安装失败，可以手动创建：

1. 打开「快捷指令」App
2. 点击右上角「+」新建快捷指令
3. 按以下步骤添加操作：

### 步骤清单

```
1. 添加操作 → 接收
   - 从: 分享表单
   - 类型: URL、纯文本

2. 添加操作 → 如果
   - 条件: 输入包含 mp.weixin.qq.com

3. 在「如果」内添加：
   - URL → 获取 URL 的内容
   - 文本 → 匹配文本
     * 模式: <title[^>]*>(.*?)<\/title>
   - 文稿 → 设定变量 (article_title)
   - 文稿 → 设定变量 (article_url)

4. 添加操作 → 其他情况
   - 文本 → 匹配文本
     * 模式: https?://[^\\s]+
   - 文稿 → 设定变量 (article_url)

5. 添加操作 → 结束如果

6. 添加操作 → 请求输入
   - 提示: 输入知萃 API 密钥
   - 输入类型: 文本

7. 添加操作 → 获取 URL 的内容
   - URL: https://api.infodigest.app/v1/contents
   - 方法: POST
   - 请求体: JSON
     * url: article_url 变量
   - 标题: Authorization
   - 值: Bearer + 空格 + 输入的 API 密钥

8. 添加操作 → 显示通知
   - 标题: 知萃
   - 正文: 内容已保存成功！
```

## 高级配置

### 自动添加标签

在快捷指令的第 7 步中，修改请求体：

```json
{
  "url": "[文章链接]",
  "tags": ["iOS快捷指令", "公众号"]
}
```

### 保存到特定收藏夹

在快捷指令中添加「从列表中选取」操作，让用户选择收藏夹。
