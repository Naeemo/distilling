# 概览

InfoDigest 是一套面向个人和助手的阅读与知识工作流，帮助你采集信息、快速理解内容、主动消化信息，并在之后重新回到它。

## 你可以在 InfoDigest 中做什么

- 保存网页链接
- 将原始文本或 Markdown 粘贴为笔记
- 以更适合浏览器的方式收集微信公众号文章
- 通过摘要、高亮和标签进行阅读
- 从一个初始想法出发，在工作区生成带参考的文章草稿
- 通过复习会话重新查看重要内容
- 在知识图谱中探索内容之间的联系

## 主要产品入口

### Web 应用

Web 应用是你资料库的主要入口。你可以用它来：

- 通过 Google、GitHub 或邮箱 magic link 登录并管理账号
- 通过链接或粘贴文本把内容送入信息中心
- 阅读并整理已保存的条目
- 在工作区里创建和编辑文章草稿
- 复习过去的内容
- 浏览知识图谱

### 浏览器扩展

浏览器扩展是收集微信公众号文章的最佳路径，因为它运行在真实浏览器标签页中，可以直接提取当前打开的页面。

### 移动端友好收集

目前这个仓库中没有独立的移动应用。更实际的移动端路径是：

- 将链接或分享文本复制到 Web 应用中
- 或者使用与你的设置匹配的 iOS 快捷指令参考流程

## 核心循环

1. 收集值得保留的内容。
2. 让 InfoDigest 将其转成信息中心里的可读条目。
3. 当你准备主动消化时，进入工作区写作、比较或综合。
4. 之后通过复习或图谱探索再次回到它。
