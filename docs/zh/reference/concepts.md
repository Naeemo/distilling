# 概念

这一页解释 InfoDigest 中面向用户的核心概念。

## 内容条目

内容条目是你资料库中的基本单位。

它可以来源于：

- 网页 URL
- 微信文章
- 粘贴文本
- Markdown

## 阅读状态

InfoDigest 使用简单的阅读状态，让你的资料库保持可排序：

- `UNREAD`
- `READING`
- `READ`

把它们当作工作流标记，而不是你是否彻底理解内容的严格评判。

## 摘要

摘要是保存条目的短版本。

你可以用它来：

- 决定现在是否要读全文
- 之后快速回忆主要观点
- 在不反复打开原文的情况下辅助复习

## 信息中心

信息中心条目是 InfoDigest 已经替你完成第一轮收纳和提纯后的信息单元。

你可以在这里：

- 快速扫一遍哪些内容值得深挖
- 重新打开某条内容继续阅读
- 挑出适合带进工作区的参考资料

## 工作区

工作区是产品里的主动思考层。

你可以在这里：

- 从一个初始想法开始
- 调用你自己的资料库作为参考
- 获得一版可编辑的文章草稿
- 继续用自己的方式重写和完善

## 高亮与笔记

高亮标记你想保留的具体片段。

笔记则是在原始内容之上加入你自己的理解。

二者结合起来，可以把被动阅读转成主动理解。

## 标签

标签是轻量级的整理标识。

你可以用它们标记：

- 项目
- 主题
- 人物
- 紧急程度
- 来源分组

## 复习

复习是产品中的记忆层。

它帮助你随着时间重新访问重要内容，而不是每次都从头重读。

## 知识图谱

知识图谱是产品中的关系层。

它帮助你看到：

- 相似性
- 张力或冲突
- 支持关系
- 共享实体
- 主题簇
