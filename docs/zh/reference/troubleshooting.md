# 故障排查

## 我保存了内容，但暂时看不到有用结果

请先检查以下几点：

- 刷新仪表盘列表
- 直接从仪表盘打开该条目
- 确认源 URL 仍然可访问
- 如果你粘贴的是文本，要记得它会被直接存储，而不是像网页那样再做提取

## 微信链接提取效果不好

微信内容的采集比普通网页提取更脆弱。

建议的兜底顺序是：

1. 使用 [浏览器扩展](/zh/platforms/browser-extension)
2. 把完整的微信分享文本粘贴进 Quick Collect
3. 如果仍然不完整，就手动保存关键文本

## 扩展提示我没有登录

预期修复方式是先回到 Web 应用，用 Google、GitHub 或邮箱 magic link 完成登录，然后重新打开扩展弹窗。

扩展依赖来自 Web 应用会话的 token 同步。

## 扩展可以打开，但保存仍然失败

请检查：

- 当前标签页是否为受支持的微信文章页面
- 打开弹窗前页面是否已经完成加载
- 文章结构是否做了非常规定制

## 我只想保存自己的笔记

使用文本模式的 Quick Collect，或在自动识别模式下粘贴纯文本或 Markdown。

这样会完全绕过页面提取。

## 我在移动端，而且不想安装任何东西

最简单的路径仍然是：

1. 复制链接或分享文本
2. 打开 Web 应用
3. 粘贴到 Quick Collect
