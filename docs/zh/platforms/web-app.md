# Web 应用

Web 应用是 InfoDigest 的核心体验。

## 你可以在那里做什么

使用 Web 应用来：

- 登录并管理会话
- 收集链接和粘贴文本
- 浏览已保存的资料库
- 打开阅读器
- 复习过去的内容
- 浏览知识图谱

## 主要页面

### 落地页

落地页介绍产品，并提供登录和注册入口。

### 仪表盘

仪表盘是你的资料库主页，也是最快的操作入口：

- 添加内容
- 按阅读状态筛选
- 搜索已保存条目
- 重新打开你之前读过的内容

### 阅读器

阅读器是你真正花时间处理内容的地方，而不只是把它归档。

### 复习

复习页面把已保存材料变成回忆练习。

### 知识图谱

图谱视图帮助你从孤立阅读走向关联理解。

## 什么时候只用 Web 应用就够了

如果你符合以下情况，Web 应用就是最好的默认选择：

- 主要保存普通网页链接
- 希望在一个地方完成阅读和整理
- 不需要依赖扩展做页面提取
