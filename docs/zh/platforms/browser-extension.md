# 浏览器扩展

浏览器扩展是在真实浏览器标签页中保存微信公众号文章的最佳路径。

## 它擅长什么

相比手动复制链接，扩展可以：

- 直接检查当前打开的页面
- 提取文章标题、作者、正文、封面图和发布时间
- 以更少的复制粘贴步骤把结果发送到 InfoDigest

## 最佳使用场景

在以下情况下使用扩展：

- 你已经在 Chrome 中打开了一篇微信文章
- 直接提取 URL 不稳定或不完整
- 你想要从阅读到保存的最低摩擦路径

## 推荐登录模型

扩展围绕 Web 到扩展的 token 同步设计。

推荐流程：

1. 先在 InfoDigest Web 应用中登录。
2. 打开扩展。
3. 让 Web 应用会话同步访问令牌。
4. 在弹窗中保存当前微信文章。

不要把扩展当成一个独立的登录入口。

## 当前打包状态

在这个仓库中，扩展目前是开发者本地加载，而不是通过浏览器商店发布。

实际安装方式是：

1. 构建扩展
2. 打开 `chrome://extensions/`
3. 启用开发者模式
4. 加载 `apps/extension` 目录

## 常见问题

- 如果扩展提示你未登录，先回到 Web 应用重新登录。
- 如果提取失败，确认当前标签页确实是微信文章页面。
- 如果弹窗能打开但无法保存，说明页面结构可能不在支持范围内。

如需更多帮助，请阅读 [故障排查](/zh/reference/troubleshooting)。
