# 知识图谱

知识图谱帮助你看到已保存条目之间的关系，而不是把每篇文章都当成孤立笔记。

## 它适合解决什么问题

当你想回答以下问题时，可以使用图谱：

- 哪些条目在讨论同一个主题
- 哪些观点彼此支持或互相矛盾
- 哪些人物、组织或主题反复出现
- 一个话题如何随时间演化

## 两种使用方式

### 图谱视图

主图谱视图适合浏览整体网络：

- 节点大小反映条目的重要性或质量
- 连线表示两个条目有关联
- 连线强度反映关联程度

### Explore 视图

Explore 视图更适合沿着图谱进行引导式探索。

它适用于：

- 深入钻研单一主题
- 追踪不同来源之间的冲突或一致
- 顺着时间线查看一系列内容
- 以实体或主题为中心展开探索

## 如何阅读图谱

图谱背后的具体数据可能会持续演进，但产品意图是稳定的：

- 相关条目应该容易被发现
- 重要条目应该在视觉上更突出
- 位置和角色应该帮助你判断下一步该读什么

## 什么时候最适合使用它

当你已经积累起有意义的资料库后，图谱会更有价值。

它尤其适合以下情况：

- 你围绕同一个主题保存了多条内容
- 你要的是综合理解，而不仅是检索
- 你正在判断哪些是第一手材料，哪些只是评论
