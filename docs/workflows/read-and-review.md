# Read and Review

InfoDigest is designed for returning to information, not just storing it.

## Read from the dashboard into the reader

The dashboard helps you scan your library by:

- reading state
- search
- tags
- recent activity

Open an item from the dashboard to move into the reader workflow.

## Reading states

Saved content generally moves through these states:

- `UNREAD`: collected but not yet processed by you
- `READING`: currently in progress
- `READ`: finished for now

These states make it easier to filter a growing library.

## Active reading tools

While reading, InfoDigest is built to support:

- highlights for important passages
- notes attached to highlights
- tags for later grouping or filtering
- summaries when available

The goal is to make each saved item easier to understand the second time you see it.

## Review workflow

The review page is where saved content becomes memory practice.

It helps you:

- see what is due today
- reveal or hide the answer content
- rate recall quality
- build a habit of spaced repetition

This is especially useful when your library contains material you want to remember, not just archive.

## A good working rhythm

1. Save content during the day.
2. Read the most important items while they are fresh.
3. Highlight or tag what matters.
4. Return through review instead of rereading everything from scratch.

## When this workflow works best

Use this path when you care about:

- retaining knowledge
- comparing related sources over time
- turning reading into reusable understanding
